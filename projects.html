PROJECTS

NEGOTIATION HACKS
Spring 2017

Created as part of a team for Wellesley HACK's Spring 2017 Hackathon. Negotiation Hacks aims to help women and minorities fight the wage gap. Users can video record themselves giving a mock pitch for why they deserve a pay raise. Negotiation Hacks then analyzes the audio and offer users advice on how they can improve their speech. It also allows users to save their recording for later reference. Negotiation Hacks is implemented using Googe Speech to Text API, Python, and Node.js.
Made with Python

SHORTCUT
Fall 2016

Created as part of a team for Wellesley HACK's Fall 2016 Hackathon. Shortcut is a website that is intended to map out the accesibility of college campuses. It also allows users to write reviews and accesibility notes about certain buildings on campus. Shortcut is implemented using a vertica database.
Made with Python.

POKEMON RUN
Summer 2016

Created as part of a group for an Intro to C++ course final project. Pokemon Run is a fun game based off of the hit phone app "Pokemon Go". The game is set from the Pokemon's point of view, and the goal is to dodge pokeballs and prevent being caught. Pokeballs are thrown at the player at random, and the more balls the player dodges, the faster they are thrown. Player is also able to save their highscore.
Made with C++.

ZOMBIE TRAILS
Spring 2016

Created as part of a pair for Data Structures Final Project. Zombie Trails is a spin off of the popular classic, Oregon Trails. It is set in a zombie apocolypse and you have just been bitten by a zombie. You have 30 days to travel to Oregon, where the cure is located, before you turn. Along the way, you meet fellow stragglers, fight zombies, and encounter both misfortune and good luck!
Made with Java.

WELP
Spring 2016

Created as part of a team for Wellesley HACK's Spring 2016 Hackathon. Welp is a rating website for Wellesley College's dining halls. It allows users to rate each of the school's dining halls and give feedback on the food served that day. Users can use this to decide where to eat!
Made with Python.
