POKEMON RUN

Title Screen

Game created in unix environment
- Start game by pressing "Ready?" button

Gameplay screen

- Goal of the game is to dodge the pokeballs.
- Player controls pikachu with up down arrows.
- Team Rocket/enemy moves randomly.
- Speed of balls and number of balls increase as game progresses.
- Score shown in bottom left.

End screen

Records highscores
- If player did not beat highscore, highscore is displayed.
